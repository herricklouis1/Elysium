# Elysium

Elysium is a minimalist, modern web project built as a foundation for internal tools, terminals, and future applications.

The project is designed to be clean, fast, and flexible — prioritizing clarity, structure, and long-term scalability.

## Live Site

https://herricklouis1.github.io/Elysium/

## Tech Stack

- HTML
- Tailwind CSS (via CDN)
- GitHub Pages

## Project Structure

- `index.html` — Main landing page
- `docs/` — GitHub Pages deployment source
- `.github/workflows/` — Automated deployment workflow

## Purpose

Elysium serves as:
- A central landing page
- A base for internal dashboards and terminals
- A long-term platform for expansion (tools, CRM, workflows)

## Status

Active development  
Deployed via GitHub Pages

## Notes

This repository intentionally avoids heavy frameworks to keep the codebase simple, readable, and easy to extend.
